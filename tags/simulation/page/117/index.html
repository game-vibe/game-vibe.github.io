Release Date: 2018/04/11

Game Description Pure Rock Crawling is an SUV simulator, the main concept of which is built around a realistic simulation of tires and suspension. Ostalgie: The Berlin Wall

Release Date: 2018/03/25

Game Description Ostalgie: The Berlin Wall is a political simulator that allows you to feel like the ruler of one of the socialist countries, such as East Germany, Romania, Poland and Bulgaria. The player’s main task is to survive until 1992. Each country has a set of problems and is driven by an individual goal. For example, the GDR wants to unite with West Germany, Romania wants to become a European version of North Korea, Bulgaria is determined to overthrow its ruler Zhivkov, and the Poles need to give a helping hand to General Jaruzelski....
