Super Buff HD

Release Date: 2023/09/29

Game Description Super Buff HD is a high-octane shooter that plunges you into a world of absurdity, where battles with bizarre bosses await. To combat this madness, an arsenal is available, including a rotating potato cannon with basketball bombs and a hammer that invariably returns to the hands after a throw....
