Generation Exile

Release Date: 2025/11/05

Game Description Generation Exile offers an atypical take on the city-building simulation genre. Instead of developing new territories and endless mining, we have to find a balance in a closed system. Progress is to create closed cycles where biological waste is transformed into nutrients or raw materials for bioplastics using anaerobic reactors. Each step in this step-by-step project requires careful planning of resources, since they cannot be replenished from the outside, and the goal is sustainability, not endless growth....
