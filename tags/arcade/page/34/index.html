Rhythm Doctor

Release Date: 2021/02/26

Game Description Rhythm Doctor is an indie game in which you need to treat patients to the rhythm of music. The music in the game has healing properties. Each patient has unusual problems based on music theory: polyrhythms, hemiols, atonality, syncopation, etc. Rhythm Doctor helps you learn music theory through gameplay. The game contains: more than 20 levels with their own stories; each chapter is a rhythmic concept; many doctors and unique patients; single and cooperative mode; about 50 different visual effects and backgrounds....
