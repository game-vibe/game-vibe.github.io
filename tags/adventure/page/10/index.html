Veterum

Release Date: 2025/11/27

Game Description Veterum is a turn-based tactical strategy with role-playing elements, the success of which directly depends on many interrelated factors. The detailed combat system forces you to take into account not only the relative position of the units, but also such parameters as morale, accumulated fatigue, the direction of the fighters' gaze, and even the level of illumination of each cell. The environment is fully interactive: you can build barricades, dig ditches, or freeze rivers to create crossings....

Quantum Loop

Release Date: 2025/11/26

Game Description Quantum Loop is a sci-fi adventure whose gameplay is based on the concept of a time loop. Any collision with an asteroid results in an immediate restart of the mission, but all information received is saved for subsequent attempts. The main task is to prevent an impending asteroid disaster that threatens the entire quadrant. The project does not offer an explicit task log, encouraging independent exploration of the world and the search for non-trivial solutions to many interrelated problems....
