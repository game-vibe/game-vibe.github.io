Haste: Broken Worlds

Release Date: 0001/01/01

Game Description Haste: Broken Worlds presents the world with a rapid action from a third person, the action of which unfolds in a setting, which is on the verge of collapse. The user will have to take on the role of Zoe, which rushes along the procedurally generated levels, trying to escape from the decay of the world surrounding it. Each of the ten fragments is a series of levels that need to be passed in a certain order, and each time the journey unfolds differently, due to the accident of procedural generation....

Lagoda Odyssey

This underground space serves as a protected area for growing a crop and storing valuable supplies far from external threats....
