Game Description#

N++ (NPLUSPLUS) is a dynamic indie platformer, each level of which is built using the simplest graphics possible, using many shades to create a memorable picture. The gamer is given a side view of a two-dimensional space, through which it is necessary to move a small man, shown schematically. The project uses the player’s skills to correctly and quickly assess the environment to make the most correct decisions. To perform precise and complex jumps, gamers will need skill and experience in this genre of video game entertainment.

It is possible to control the gameplay using a controller, as well as pass prepared tests in cooperative mode using one monitor screen. Among the multiplayer pastime modes, there is a chance to take part in battles against friends in a noisy company on specially prepared maps, which are absent in the main tests. The project supports simultaneous play by 4 people. The statistics of each of them are included in the leaderboard, which marks the best results of gamers from around the globe.

The main feature of the indie platformer in retro style is a gigantic number of levels of varying difficulty - 4340 stages. All of them are created by hand and have a different landscape. In addition, the game client has a built-in map editor that allows you to create unique things and share them with the community, grading the finished result. The gameplay takes place under a selected soundtrack, which includes 63 compositions from electronic music performers of varying degrees of popularity.
