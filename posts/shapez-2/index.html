Game Description#

Shapez 2 is a simulator about the construction of factories with a top view, in which you need to automate the process of processing geometric shapes. Players must build and optimize massive production facilities to solve increasingly complex logistics problems. Gameplay involves cutting, rotating, stacking, painting and delivering shapes to unlock advanced technologies and expand enterprise capabilities.

What’s unique is that each object is a separate set of building blocks that can be disassembled, folded, painted and put back together in special ways. Can be built in multiple interconnected layers, allowing for compact and streamlined factories. In addition, there are no enemies or time limits, and all buildings are free. Gamers start their activities with one asteroid and constantly expand, building new platforms. As you progress, layouts open up and they can grow from a single belt into a huge space factory with multiple stations connected by trains. It is also possible to design modular platforms with blueprint creation mechanics for large-scale construction, placing hundreds of buildings with one click.

One of the most significant features is 3D visualization, which works smoothly even in giant factories. Performance is a top priority for the developers who rewrote the game in a new engine. Overall, Shapez 2 is a challenging yet easy to learn simulation game that provides players with endless possibilities for automating shape processing. With no penalties for errors, you have complete control over your projects, allowing you to create the most efficient mold production lines.
