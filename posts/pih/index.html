Game Description#

In Pih, the main goal is to clear locations by collecting all available food. Players progress through each level while avoiding hostile pigs as their main opponents. The project contains 64 different stages, thematically grouped into four seasons. Each seasonal block ends with a battle with one of four bosses, marking a clear progression in the playthrough structure.

Pih’s concept prioritizes continuous gameplay, eschewing narrative elements like story or cutscenes. This approach extends to the technical implementation, which eliminates loading screens between levels to create a smooth flow between challenges. Gamer interaction is entirely focused on the core mechanics of collecting items and dodging in a platform environment. Defeated opponents react using ragdoll physics, adding dynamism to encounters.

The project is designed to be played both on regular PC monitors and on virtual reality devices. Progression forward is sequential - to unlock the next location you must successfully complete the previous one. In addition to the main seasonal content, a separate set of Halloween-themed levels offers new challenges. The atmosphere is defined by a straightforward, task-oriented structure: each zone is its own puzzle involving movement and finding objects.
