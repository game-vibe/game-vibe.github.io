Game Description Quantum Loop is a sci-fi adventure whose gameplay is based on the concept of a time loop. The main task is to prevent an impending asteroid disaster that threatens the entire quadrant. The project does not offer an explicit task log, encouraging independent exploration of the world and the search for non-trivial solutions to many interrelated problems....
